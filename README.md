# shared/

Shared data and pipeline scripts used by both `panel-member` and `bridge-coder`.

---

## Directory structure

```
shared/
├── pipeline/               # Download scripts
│   ├── download_reports.py       # HTML sources: SD, FH, IRFR
│   ├── download_reports_pdf.py   # SD PDF download + text extraction (fallback)
│   └── check_section_coverage.py # Audit section coverage across processed-text
├── processed-text/         # Plain-text reports (downloaded locally, not committed)
│   ├── state-dept/{year}/{slug}.txt
│   ├── freedom-house/{year}/{slug}.txt
│   └── irfr/{year}/{slug}.txt
├── source-docs/            # Raw PDFs (downloaded locally, not committed)
│   └── state-dept/{year}/{slug}.pdf
├── vdem-data/              # V-Dem data files (not committed — see below)
│   ├── panel_means.csv     # Generated by generate_vdem_data.R
│   └── human_ratings.csv   # Generated by generate_vdem_data.R
├── generate_vdem_data.R    # Generates panel_means.csv and human_ratings.csv from V-Dem v15
├── requirements.txt        # Python dependencies for pipeline scripts
└── venv/                   # Python virtual environment (not committed)
```

---

## Setup

```bash
cd shared/
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

---

## Downloading source documents

Run all scripts from the `shared/` directory with the venv active.

**State Dept Human Rights Reports (HTML):**
```bash
python3 pipeline/download_reports.py --source state-dept --year 2019
```

**Freedom House Freedom in the World (HTML):**
```bash
python3 pipeline/download_reports.py --source freedom-house --year 2019
```

**IRFR — International Religious Freedom Reports (executive summary only):**
```bash
python3 pipeline/download_reports.py --source irfr --year 2019
```

**All three sources at once:**
```bash
python3 pipeline/download_reports.py --source all --year 2019
```

**Subset of countries:**
```bash
python3 pipeline/download_reports.py --source state-dept --year 2019 --countries nigeria kenya
```

**SD PDFs (fallback when HTML extraction fails):**
```bash
# Download PDFs only
python3 pipeline/download_reports_pdf.py --year 2019

# Download PDFs and extract text
python3 pipeline/download_reports_pdf.py --year 2019 --extract

# Extract text from already-downloaded PDFs
python3 pipeline/download_reports_pdf.py --year 2019 --extract-only
```

## V-Dem data

`panel_means.csv` and `human_ratings.csv` are generated from the V-Dem v15
coder-level dataset. To regenerate:

1. Download the V-Dem v15 coder-level CSV from the V-Dem Institute and place it
   at `vdem-data/V-Dem-Coder-Level-v15_csv/`.
2. Run `Rscript generate_vdem_data.R` from `shared/`.

Neither the raw V-Dem data nor the generated CSVs are committed to git.
